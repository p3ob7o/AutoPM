This project is about automating entirely the development of LeanDomainSearch.com by implementing a team of managed agents that run in the Anthropic cloud, with each agent having a different role:

- Product Manager Agent that owns the project
- QA Agent that tests the site continuously
- Customer Interview Agent that interviews users once they purchase domains
- Support Agent that helps users
- Design Agent
- Engineering Agent
- Marketing Agent
- Project Manager Agent to organize tasks.

All these agents perform all the operations that the PM Agent requires.

This connects to the research project that is in [research/ManagedAgents](../../research/ManagedAgents/README.md).
